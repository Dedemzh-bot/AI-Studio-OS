# 角色私人宿舍 - 宏观设计草案

## 一、系统概述与设计愿景
- **核心目的**：为玩家提供与已获取角色进行深度、私密、沉浸式互动的专属空间，强化“陪伴感”与“角色归属感”，是游戏“ML向情感羁绊”支柱的核心落地系统。
- **系统定位**：高价值付费内容（角色级买断）+ 长线情感沉淀容器。非核心战斗循环，但承担核心付费转化与留存任务。
- **设计原则**：每个角色拥有独立、不可复用的专属宿舍场景，买断后永久解锁，强调“独占感”与“私密性”。

## 二、核心规则与玩法机制
- **解锁规则**：玩家获得某角色后，宿舍系统基础入口开放，但该角色的专属宿舍场景需通过“角色宿舍买断券”（付费道具，定价约18元/角色）解锁。
- **核心玩法循环**：
  1.  **进入宿舍**：从主界面或角色详情页进入已买断角色的私人宿舍。
  2.  **自由探索**：在3D宿舍场景中自由移动视角，点击场景中的互动点（如沙发、床、窗户、衣柜）。
  3.  **触发互动**：每个互动点对应1-2种角色专属动作/语音/表情反馈（如坐在沙发上撩头发、躺在床上伸懒腰、在窗边眺望并回头微笑）。
  4.  **好感度反馈**：互动频率与好感度等级挂钩，高好感度解锁更亲密动作（如角色主动靠近镜头、脸颊微红、特殊对话）。
  5.  **换装展示**：宿舍内角色可切换玩家拥有的所有该角色皮肤，并支持自由旋转/缩放视角进行“鉴赏模式”。
- **核心机制**：
  - **互动点系统**：每个宿舍预设5-8个固定互动点，每个互动点有独立冷却时间（约5分钟），防止刷取疲劳。
  - **好感度联动**：互动时增加微量好感度（每日有上限），好感度等级影响互动动作的“亲密程度”与语音内容。
  - **摄影模式**：玩家可进入“摄影模式”，自由调整相机角度、滤镜、角色表情（预设几种），截图保存至本地。

## 三、表现层与角色展示联动 (核心强制项)
- **场景设计**：每个角色宿舍风格与角色性格/背景强绑定（如：冷淡系角色-简约现代风；活泼系角色-温馨可爱风），场景内包含大量角色专属小物件（如喜欢的玩偶、乐器、书籍），强化沉浸感。
- **角色模型展示**：
  - 宿舍内角色模型精度与战斗模型一致，但增加“待机动作”多样性（如看书、玩手机、哼歌）。
  - 角色始终面向玩家镜头，跟随玩家视角转动头部与视线（“视线跟随”机制）。
- **特殊机位与擦边互动**：
  - **低角度机位**：当玩家将视角拉低至角色腰部以下时，角色会做出“害羞遮挡”或“轻踢”动作，并触发特殊语音（如“分析员，看哪里呢~”），严格控制在擦边暗示范围内。
  - **背部视角**：角色转身时，镜头自动聚焦至臀部与腰部曲线，配合布料物理抖动。
  - **触摸反馈**：玩家可点击角色身体特定区域（头、肩、手），触发对应语音与动作（如摸头-角色眯眼享受；摸手-角色反握玩家手指）。
- **物理表现**：头发、胸部、裙摆等部位应用高规格软体物理与Jiggle Physics，在角色转身、坐下、躺下时表现明显。

## 四、经济循环与商业化埋点
- **核心付费点**：
  - **角色宿舍买断券**：定价18元/个，直购解锁单个角色专属宿舍。可放入限时礼包（如“新手宿舍礼包”：3张券+少量资源，定价45元）。
  - **宿舍装饰扩展包**（可选）：后续可推出通用装饰物（如壁纸、地毯、灯光颜色），通过小额付费或活动获取，不强制。
- **免费资源产出**：
  - 每日首次进入任意已买断宿舍，赠送少量“好感度礼物”（用于提升好感度）。
  - 每周完成宿舍互动任务（如“与3个不同角色互动”），赠送1张“宿舍体验券”（可临时解锁一个未买断角色的宿舍24小时，刺激购买转化）。
- **经济循环**：买断券为一次性消耗品，无后续持续付费压力，符合“买断制”定位。好感度礼物为养成资源，可通过日常玩法产出，维持活跃。

## 五、旧系统与数据联动 (强依赖)
- **角色系统**：宿舍解锁状态与角色持有状态绑定。角色被分解/遣散时，已买断的宿舍保留（角色回归后仍可用）。
- **好感度系统**：宿舍互动产出好感度，与现有好感度等级、剧情解锁、属性加成系统共享同一数值体系。
- **皮肤系统**：宿舍内角色换装直接调用玩家已拥有的该角色皮肤资源，无需额外付费。
- **主界面入口**：在角色详情页增加“进入宿舍”按钮，在主界面增加“宿舍”快捷入口（仅显示已买断角色列表）。

## 六、待确认风险与追问
- **技术风险**：每个角色独立3D宿舍场景（含互动点、物理、AI行为）的开发成本较高，需评估初期上线角色数量（建议首发3-5个角色宿舍，后续版本持续追加）。
- **付费接受度**：18元/角色的买断定价是否与目标用户付费习惯匹配？需通过小规模测试验证转化率。
- **内容消耗速度**：当前每个宿舍5-8个互动点，玩家可能在1-2周内体验完所有内容。需规划后续版本更新节奏（如每版本新增1-2个角色宿舍 + 旧宿舍新增互动点）。
- **尺度合规**：触摸反馈与低角度机位需与法务/审核团队提前对齐，确保不触碰R18红线。
