# 温泉中心 - 宏观设计草案

## 一、系统概述与设计愿景

- **系统定位**：温泉中心是宿舍系统的扩展场景，属于纯陪伴/观赏/轻度互动类子系统。不产出战斗数值，不涉及抽卡资源，核心目的是提供角色在“温泉”场景下的沉浸式展示与互动。
- **前置条件**：玩家必须已解锁宿舍系统，且至少拥有1名角色。
- **入口**：从宿舍场景内新增“温泉”入口按钮，点击后加载温泉中心场景。
- **数据归属**：所有温泉中心数据（角色刷新、互动记录、小游戏进度）归属于玩家账号全局存档，与角色好感度系统弱关联（仅记录互动行为，不直接产出好感度经验）。
- **边界**：温泉中心不产出任何抽卡资源、战斗属性、养成材料。产出仅限于：角色特殊语音/表情/动作的解锁、温泉中心专属的装饰/道具（纯观赏）。

## 二、核心规则与玩法机制

### 2.1 角色刷新规则
- **刷新逻辑**：温泉中心每次进入时，从玩家已拥有的角色池中随机抽取1名角色，以“泡澡”状态出现在场景中。
- **刷新权重**：可配置基础权重（所有角色均等） + 当日首次进入时，可设置“今日特惠角色”（若玩家拥有该角色，则必定刷新该角色，每日限1次）。
- **角色状态**：角色处于“泡澡”状态，模型为泳装/浴巾装扮（需美术资源），场景中角色半身浸入水中，水面以上可见上半身、手臂、头发等。
- **退出/切换**：玩家可主动点击“换人”按钮，消耗少量基础货币（如金币）重新随机刷新角色。每日免费刷新次数上限为3次。

### 2.2 互动小游戏 - 水枪射击靶子（轻度休闲解压类）
- **玩法描述**：玩家使用水枪（屏幕准星）瞄准场景中随机出现的漂浮靶子（如：水球、浮标、鸭子），点击射击。命中后靶子爆裂/消失，播放角色特殊反馈（如：被水花溅到后的惊讶/害羞表情+语音）。
- **核心循环**：进入小游戏 → 场景中刷新10个靶子（随机位置） → 玩家在30秒内射击 → 命中数达到阈值（如8/10）触发角色特殊奖励（解锁专属表情/语音） → 小游戏结束，返回温泉中心主场景。
- **奖励**：命中数达标后，解锁该角色在温泉中心场景下的一个特殊互动动作/语音（永久解锁，可在图鉴中回看）。不产出任何数值资源。
- **限制**：每个角色每日仅可触发1次小游戏奖励。小游戏本身无次数限制，但奖励仅每日1次。

### 2.3 互动小游戏 - 捞水球（轻度休闲解压类）
- **玩法描述**：场景中出现一个水池，水面漂浮若干彩色水球。玩家通过点击/拖拽操作，将水球捞至岸边指定区域。每次成功捞取，水球破裂并播放角色反馈（如：开心鼓掌、微笑）。
- **核心循环**：进入小游戏 → 场景中出现5个水球 → 玩家需在20秒内捞取至少4个 → 成功则触发角色特殊奖励（解锁专属表情/语音） → 小游戏结束。
- **奖励**：同水枪射击，解锁该角色在温泉中心场景下的一个特殊互动动作/语音（永久解锁）。
- **限制**：每个角色每日仅可触发1次小游戏奖励。水枪射击与捞水球共享每日奖励次数（即：玩家只能从两个小游戏中各获得1次奖励，或选择只玩其中一个）。

## 三、表现层与角色展示联动（核心强制项）

### 3.1 场景与运镜
- **场景设计**：日式露天温泉风格，背景为山景/竹林，水面有蒸汽粒子特效。角色半身浸入水中，水面以上可见上半身、手臂、头发。模型为泳装/浴巾装扮，头发有湿发效果（需美术资源）。
- **默认镜头**：第三人称近肩视角，略微俯视（模拟玩家坐在池边视角）。镜头可自由旋转/缩放，但限制最低角度（防止穿模或过度暴露）。
- **特殊机位**：小游戏触发时，镜头自动推近至角色面部特写，展示表情变化。小游戏结束后镜头平滑复位。

### 3.2 物理反馈
- **水面物理**：角色移动/互动时，水面产生波纹与涟漪。水球/靶子被击中时，水花溅射特效。
- **角色物理**：头发、浴巾边缘有物理抖动。被水花溅到时，角色有轻微后仰/躲闪动画。
- **表情系统**：角色根据互动类型播放对应表情（开心、惊讶、害羞、微笑）。所有表情严格控制在“可爱/害羞”范围内，禁止任何性暗示表情。

### 3.3 UI透明化
- 所有UI组件（小游戏计时器、得分、按钮）采用半透明毛玻璃设计，不遮挡角色主体。小游戏进行时，UI自动隐藏，仅保留准星/操作提示。
- 玩家可一键隐藏所有UI，进入纯观赏模式。

### 3.4 擦边底线
- 角色泳装/浴巾装扮严格符合游戏分级标准（不露点、不透视）。水面高度控制在胸部以下，不展示隐私部位。
- 角色反馈（被水花溅到、被瞄准）均为“可爱/害羞”反应，禁止任何性暗示动作或台词。
- 触摸互动：温泉中心不开放触摸互动功能（仅观赏与小游戏）。

## 四、经济循环与商业化埋点

### 4.1 免费产出
- 每日首次进入温泉中心：免费获得1次“今日特惠角色”刷新机会（必定刷新指定角色）。
- 每日免费刷新次数：3次（消耗基础货币可额外购买）。
- 小游戏奖励：每日每个角色限1次，解锁专属表情/语音（永久解锁，不消耗任何资源）。

### 4.2 付费埋点
- **温泉中心专属皮肤**：在商城中出售温泉场景专用的泳装/浴巾皮肤（比基础泳装更精致/尺度略大，但仍符合分级）。购买后可在温泉中心中穿戴。
- **刷新券**：付费道具，使用后可立即刷新角色（不消耗每日免费次数）。
- **小游戏次数重置券**：付费道具，使用后可重置某个角色的小游戏奖励次数（再次获得解锁内容的机会，但解锁内容不重复，仅用于补漏）。

### 4.3 资源循环
- 温泉中心不产出任何抽卡资源、战斗属性、养成材料。
- 所有产出均为纯观赏/陪伴向内容（表情、语音、动作解锁）。
- 基础货币消耗：刷新角色、购买小游戏次数重置券。

## 五、旧系统与数据联动

### 5.1 宿舍系统
- 温泉中心作为宿舍系统的扩展场景，入口位于宿舍场景内。
- 宿舍系统的“换装”功能可切换温泉中心中角色的泳装皮肤（需已购买）。
- 宿舍系统的“拍照模式”在温泉中心中同样可用（但限制镜头角度）。

### 5.2 角色获取系统
- 温泉中心角色池 = 玩家已拥有的所有角色。未获取的角色不显示。
- 新角色获取后，自动加入温泉中心角色池（无需额外解锁）。

### 5.3 图鉴系统
- 温泉中心解锁的特殊表情/语音/动作，自动同步至图鉴系统，玩家可在图鉴中回看。
- 图鉴中新增“温泉中心”分类，展示所有已解锁内容。

### 5.4 好感度系统
- 温泉中心互动行为不直接产出好感度经验。
- 但温泉中心解锁的特殊内容（表情/语音）可在宿舍系统中使用（如：触摸互动时触发温泉中心专属语音）。

## 六、待确认风险与追问

### 6.1 性能风险
- 温泉中心场景需同时加载角色模型（泳装/浴巾）、水面物理、蒸汽粒子、小游戏逻辑。建议初期仅允许1个角色同时出现，后续根据性能优化情况开放多角色同池。
- 若设备性能不足，可降级水面物理与粒子特效。

### 6.2 内容生产量
- 每个角色需要独立制作：泳装/浴巾模型、湿发效果、5-8条温泉专属语音、3-5个表情、2个小游戏反馈动画。
- 首批仅覆盖3-5个高人气角色，后续角色分批更新。未覆盖角色显示“即将上线”提示。

### 6.3 小游戏平衡性
- 水枪射击与捞水球的难度需适中，确保大部分玩家可在1-2次尝试内完成。若玩家多次失败，可提供“跳过”选项（消耗基础货币）。
- 小游戏奖励为永久解锁内容，需确保每个角色有足够多的专属内容可供解锁（建议每个角色至少3个解锁位）。
