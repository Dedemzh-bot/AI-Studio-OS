# 冰霜新星技能数据验收表

## 字段说明

| 字段名 | 含义 | 数据类型 | 取值范围/约束 | 必填 | 防漏洞措施 |
|--------|------|----------|---------------|------|------------|
| skill_id | 技能唯一标识 | 字符串 | 必须符合 `sk_属性_名称` 格式，例如 sk_ice_frost_nova | 是 | 强制命名规范，防止ID冲突 |
| skill_name | 技能显示名称 | 字符串 | 长度 1-64 字符 | 是 | 限制长度，防止溢出 |
| description | 技能描述 | 字符串 | 长度 1-256 字符 | 是 | 限制长度，防止描述过长 |
| base_damage | 初始基础伤害 | 整数 | 0-100 | 是 | 限制最大伤害100，防止前期数值爆炸 |
| freeze_duration | 冻结持续时间 | 浮点数 | 0-10 秒 | 是 | 限制最大冻结时间，防止无限控制 |
| radius | 技能范围半径 | 浮点数 | 0-500 单位 | 是 | 限制最大范围，防止全屏技能 |
| trigger_source.entity_id | 触发实体ID | 字符串 | 非空字符串 | 是 | 追踪技能来源，防止无主技能 |
| trigger_source.position | 触发位置坐标 | 对象 | 包含 x, y, z 三个浮点数 | 是 | 记录精确位置，用于范围判定和漏洞追踪 |
| trigger_source.timestamp | 触发时间戳 | 浮点数 | ≥0 | 是 | 记录时间，用于时序分析和防止重放攻击 |

## 设计防漏洞措施

1. **因果追踪**：通过 `trigger_source` 对象完整记录技能触发的实体、位置和时间，可追溯任何技能事件的来源，防止地图边缘爆炸等异常情况导致无限刷怪。

2. **数值平衡**：`base_damage` 上限设为 100，符合全局数值规范，防止前期伤害过高破坏体验。

3. **范围限制**：`radius` 上限 500 单位，防止技能范围无限扩大导致全屏攻击。

4. **时间控制**：`freeze_duration` 上限 10 秒，防止无限控制敌人。

5. **命名规范**：`skill_id` 强制正则校验，确保符合 `sk_属性_名称` 格式。

6. **禁止额外字段**：所有对象均设置 `additionalProperties: false`，防止注入非法数据。
