### 暗影步 (ShadowStep) 技能配置验收表

| 字段名 | 含义 | 取值范围/格式 | 必填 | 防漏洞措施 |
| :--- | :--- | :--- | :--- | :--- |
| **skill_id** | 技能的唯一ID | 字符串，必须符合 `sk_属性_名称` 格式，例如 `sk_dark_shadow_step` | **是** | 强制全局命名规范，避免ID冲突。 |
| **skill_name** | 技能显示名称 | 字符串，长度1-64字符 | **是** | 防止名称为空或过长。 |
| **skill_description** | 技能描述 | 字符串，最长256字符 | 否 | 防止描述文本过长。 |
| **base_damage** | 基础伤害值 | 整数，范围 1-100 | **是** | **核心数值平衡**：根据公司规范，伤害上限为100，防止前期数值爆炸。 |
| **displacement_distance** | 位移距离 | 数字（浮点数），必须 >= 0 | **是** | 防止负距离导致的逻辑错误。 |
| **trigger_source_id** | 触发来源ID | 字符串，只能包含字母、数字、下划线 | **是** | **因果追踪**：明确记录是谁/什么触发了这个技能。防止“凭空”触发的漏洞（例如，一个没有来源的爆炸）。 |
| **causal_chain_tag** | 因果链标签 | 字符串，只能包含字母、数字、下划线 | **是** | **防无限循环**：所有由同一个初始事件（如一次攻击）产生的效果，都必须带上相同的标签。系统在处理效果时，如果检测到同一个标签被重复触发，可以中断链条，防止“地图边缘爆炸 -> 触发暗影步 -> 位移到另一个边缘 -> 再次爆炸 -> ...”这样的无限循环。 |
| **is_passive** | 是否为被动技能 | 布尔值（true/false），默认为false | 否 | 明确技能类型，防止被动技能被错误地主动使用。 |
| **cooldown_time** | 冷却时间 | 数字（浮点数），必须 >= 0，默认为0 | 否 | 防止负冷却时间。 |
