# 项目宪法：Project T

## 1. 核心愿景 (Core Vision)
本项目是一款以"极度媚宅的角色表现"为核心驱动力，结合"高品质TPS射击体验"的二次元美少女抽卡养成RPG。
游戏的最高准则是：**所有系统设计、UI交互和战斗机制，必须无条件为"展现女性角色的魅力、身材与陪伴感"服务。**

## 2. 核心设计支柱 (Game Pillars)
* **极致的表现层 (Extreme Presentation)：** * 强调高精度的 3D 模型、细腻的材质（汗水、布料透视）。
    * 战斗与大厅必须包含精心设计的运镜（如后背视角、蹲掩体视角的特殊关照）。
    * 搭载深度定制的物理引擎（头发飘动、高规格的软体碰撞/Jiggle Physics）。
* **ML向的情感羁绊 (Master-Love Focus)：** * 玩家扮演绝对核心的"分析员"，所有女性角色对玩家抱有极高的好感与依赖。
    * 包含丰富的私密互动场景（如基地宿舍、短信聊天、专属好感度剧情、触摸反馈）。
* **核心设计：在玩法系统，活动系统，成长等系统设计时尽可能考虑有展示角色模型或者立绘，来凸显女角色。
* **降维的射击体验 (Accessible TPS)：** * 战斗是为了让玩家看角色开枪的帅气/性感姿态，而不是折磨玩家。
    * 必须包含辅助瞄准、掩体自动规避等轻量化设计，数值碾压可以大幅降低操作门槛。

## 3. 设计红线与绝对禁忌 (Redlines - 碰则打回)
* **【禁忌 1】严禁出现任何可操控的男性角色（男自机）。** 游戏内的核心交互对象必须 100% 为女性。
* **【禁忌 2】严禁出现任何可能导致"NTR"或"角色与非玩家NPC有暧昧"的剧情与设定。**
* **【禁忌 3】严禁设计大面积遮挡角色模型的 UI。** 任何战斗 HUD 和结算界面，必须把屏幕最核心的位置留给角色的身段展示。
* **【禁忌 4】拒绝重度硬核 PVP。** 尽量以 PVE 割草、竞速或非对称休闲玩法为主，避免玩家因挫败感产生流失。
* **【禁忌 5】🔞 尺度分级底线（生死红线）：** 确认本项目为"擦边媚宅"定位，绝非纯成人黄油。**严禁**在任何系统、剧情、玩法中出现真正的性行为、做爱、直接露点（R18）等设计。所有角色玩法与互动必须严格控制在"灰色地带"——仅允许使用性暗示、极其暧昧的氛围、擦边镜头（如特写、物理抖动）或ASMR级别的听觉刺激。点到为止，给玩家留足遐想空间，但物理上绝对不可越界。

## 4. 商业化倾向 (Monetization Strategy)
* **核心付费点：** 角色抽卡（Gacha）、专属武器抽卡。
* **表现付费点：** 尺度更大的高质量角色皮肤（包含专属 Live2D、大厅特殊互动姿势、特殊爆衣/战损特效）。
* **系统要求：** 任何新增的日常/周常玩法，都必须产出少量抽卡资源，以维持平民玩家的活跃度。
